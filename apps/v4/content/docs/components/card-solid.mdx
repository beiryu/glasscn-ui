
## API Reference

### CardSolid

| Prop        | Type                                                     | Default     | Description                  |
| ----------- | -------------------------------------------------------- | ----------- | ---------------------------- |
| `title`     | `string`                                                 | -           | The title/label of the card  |
| `value`     | `string \| number`                                       | -           | The main value to display    |
| `icon`      | `"users" \| "user-plus" \| "trending-up" \| "activity"`  | -           | Optional icon name           |
| `variant`   | `"default" \| "success" \| "warning" \| "danger"`        | `"default"` | Visual variant of the card   |
| `size`      | `"default" \| "sm" \| "lg"`                              | `"default"` | Size of the card             |
| `state`     | `"default" \| "hover" \| "disabled"`                     | `"default"` | State of the card            |
| `trend`     | `{ value: string, label: string, isPositive?: boolean }` | -           | Optional trend indicator     |
| `disabled`  | `boolean`                                                | `false`     | Whether the card is disabled |
| `className` | `string`                                                 | -           | Additional CSS classes       |

### Preset Components

The component also exports preset components for common use cases:

- `ActiveCreatorsCardSolid` - Pre-configured for active creators with Users icon
- `NewSignupsCardSolid` - Pre-configured for new signups with UserPlus icon

```tsx showLineNumbers
import {
  ActiveCreatorsCardSolid,
  NewSignupsCardSolid,
} from "@/components/ui/card-solid"

<ActiveCreatorsCardSolid value="07" />
<NewSignupsCardSolid value="03" />
```
