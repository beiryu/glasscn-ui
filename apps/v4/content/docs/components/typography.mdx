```tsx showLineNumbers
<Title textColor="ghostwhite">Title with Ghostwhite Color</Title>

<Body textColor="ghostwhite">Body with Ghostwhite Color</Body>

```

### Custom Element

You can use the `as` prop to render a different HTML element:

```tsx showLineNumbers
<Typography variant="title" as="h1">
  Custom Title as h1
</Typography>

<Typography variant="subheading" as="h2">
  Custom Subheading as h2
</Typography>
```

### With Custom Classes

```tsx showLineNumbers
<Typography variant="body" className="opacity-num-0_5">
  Body text with custom opacity
</Typography>
```

## API Reference

### Typography

The main `Typography` component that accepts variant and textColor props.

| Prop       | Type                                                                              | Default     |
| ---------- | --------------------------------------------------------------------------------- | ----------- |
| `variant`  | `"title" \| "subheading" \| "body" \| "body-small"`                             | `"body"`    |
| `textColor`| `"default" \| "ghostwhite" \| "muted"`                                           | `"default"` |
| `as`       | `React.ElementType`                                                               | `"p"`       |
| `asChild`  | `boolean`                                                                         | `false`     |

### Title

Convenience component for title variant. Defaults to `h2` element.

| Prop       | Type                                                                              | Default     |
| ---------- | --------------------------------------------------------------------------------- | ----------- |
| `textColor`| `"default" \| "ghostwhite" \| "muted"`                                           | `"default"` |
| `as`       | `"h1" \| "h2" \| "h3" \| "h4" \| "h5" \| "h6"`                                   | `"h2"`      |

### Subheading

Convenience component for subheading variant. Defaults to `h3` element.

| Prop       | Type                                                                              | Default     |
| ---------- | --------------------------------------------------------------------------------- | ----------- |
| `textColor`| `"default" \| "ghostwhite" \| "muted"`                                           | `"default"` |
| `as`       | `"h1" \| "h2" \| "h3" \| "h4" \| "h5" \| "h6"`                                   | `"h3"`      |

### Body

Convenience component for body variant. Defaults to `p` element.

| Prop       | Type                                                                              | Default     |
| ---------- | --------------------------------------------------------------------------------- | ----------- |
| `textColor`| `"default" \| "ghostwhite" \| "muted"`                                           | `"default"` |

### BodySmall

Convenience component for body-small variant. Defaults to `p` element.

| Prop       | Type                                                                              | Default     |
| ---------- | --------------------------------------------------------------------------------- | ----------- |
| `textColor`| `"default" \| "ghostwhite" \| "muted"`                                           | `"default"` |

## Typography Variants

| Variant      | Font Size | Line Height | Font Weight | Use Case              |
| ------------ | --------- | ----------- | ------------ | ---------------------- |
| `title`      | 24px      | 28px (1.75rem) | SemiBold (600) | Main page titles, section headers |
| `subheading` | 18px      | 26px (1.625rem) | Medium (500) | Subsection titles, card headers |
| `body`       | 16px      | 24px (1.5rem) | Medium (500) | Main content text, paragraphs |
| `body-small` | 14px      | 20px (1.25rem) | Regular (400) | Secondary text, captions, labels |

All variants use:
- **Font**: Inter (`font-inter`)
- **Letter Spacing**: -0.01em (`tracking-num--0_01`)
